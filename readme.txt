Monopoly game created following the instructions from

https://schuchert.wikispaces.com/Monopoly%28r%29

It is written in C# and uses ASP.NET MVC 5.

This just a basic program, it is not picture perfect.

This project is a demonstration of OOP in practice:
	Model Designing with multiple levels of inheritance
	Polymorphism in action
Advanced usage of C#
	Passing functions as parameters
	Usage of Reflection
	Extensive use of LINQ functions
It also shows a demonstration of TDD
